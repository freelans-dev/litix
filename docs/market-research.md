## 1. Resumo Executivo

O mercado brasileiro de legaltech apresenta oportunidade significativa para uma plataforma de monitoramento processual B2B com diferencial multi-provider. Com 1,3M+ advogados ativos, 600+ startups legaltech e adocao tecnologica acelerada (55,1% ja usam IA), o timing e favoravel para entrada com produto API-first.

**Dimensionamento:**
- **TAM:** R$ 270-470M (monitoramento + consulta processual BR)
- **SAM:** R$ 50-90M (escritorios digitalmente alcancaveis, horizonte 3 anos)
- **SOM:** R$ 2-7M ARR (Y1-Y3, capturando 2-7% do SAM)

**Diferencial competitivo do Litix:** Unica solucao com redundancia multi-provider (DataJud + Codilo + Escavador + Judit + Predictus), transparencia de fonte, e entrega via webhook push — gaps nao cobertos pelos incumbentes.

## 6. Panorama Competitivo

### 6.1 Estrutura do Mercado

O mercado e fragmentado com dois grandes incumbentes de marca (JusBrasil e Escavador) que dominam o segmento de pesquisa e monitoramento basico, e um segundo grupo de softwares de gestao juridica (Projuris, eLaw, Astrea/Aurum) que oferecem monitoramento como feature dentro de uma suite mais ampla. Nao existe um player dominante no segmento especifico de "API processual B2B" para escritorios de medio/grande porte.

### 6.2 Analise dos Principais Players

**JusBrasil**
- Descricao: Maior plataforma juridica do Brasil; repositorio de jurisprudencia, legislacao e monitoramento processual
- Forcas: Marca fortissima, base de dados historica imensa, versao free com grande adocao, API B2B
- Fraquezas: Produto massificado, pouca customizacao, SLA nao divulgado, API para B2B menos conhecida
- Foco de cliente: Amplo (todos os advogados, estudantes, cidadaos)
- Precificacao: Free tier robusto; planos pagos nao transparentes publicamente para o B2B/API

**Escavador**
- Descricao: Plataforma focada em dados judiciais estruturados, com forte produto de monitoramento e API bem documentada
- Estimativa de crescimento: 50% ao ano (declarado pela empresa)
- Forcas: API V1/V2 bem documentada, monitoramento de Diarios Oficiais de todo o Brasil, planos acessiveis
- Fraquezas: Marca menos reconhecida que JusBrasil entre advogados mais tradicionais
- Precificacao: A partir de R$ 9,90/mes (15 processos); API pay-per-use

**Astrea (Aurum)**
- Descricao: Software de gestao juridica com modulo de monitoramento processual integrado
- Forcas: Suite completa, marca Aurum estabelecida, UX moderna, plano light gratuito
- Fraquezas: Monitoramento e feature, nao produto core; sem posicionamento de API
- Precificacao: Light gratuito (40 processos); pagos a partir de R$ 21,64/mes

**Projuris ADV (SAJ)**
- Descricao: Software juridico enterprise com monitoramento, gestao de processos e integracao
- Forcas: Produto maduro, cliente enterprise, integracao com SAJ, SLA robusto
- Fraquezas: Precificacao enterprise complexa; curva de aprendizado maior
- Precificacao: A partir de R$ 197,27/mes; enterprise sob consulta

**eLaw**
- Descricao: Plataforma de gestao juridica com foco em IA e automacao
- Forcas: Posicionamento de IA, integracao com PJe
- Precificacao: Sob consulta (modelo enterprise)

## 7. Analise da Industria (Porter's Five Forces)

**Poder dos Fornecedores: MEDIO**
O principal "fornecedor" e o Poder Judiciario (dados via DataJud API publica e scraping). A API publica do CNJ e gratuita mas tem limites de rate e cobertura incompleta. Construir camada de cache e agregacao reduz dependencia.

**Poder dos Compradores: MEDIO-ALTO**
Advogados autonomos tem alto poder por serem price-sensitive e poder migrar facilmente. Escritorios medios tem mais lock-in por integracao.

**Rivalidade Competitiva: MEDIA**
Dois grandes incumbentes de marca mas o segmento de API B2B para medio porte nao tem lider claro. Diferenciacao possivel por qualidade de dados, SLA e UX.

**Ameaca de Novos Entrantes: MEDIA**
Barreiras: (1) cobertura tecnica de todos os tribunais, (2) reputacao no mercado conservador juridico, (3) dados historicos para jurimetria.

**Ameaca de Substitutos: MEDIA**
Substitutos: consulta manual nos portais, scraping generico, estagiarios. O custo-beneficio do SaaS vence em escala.

**Estagio de Adocao Tecnologica:** Early Majority — janela de entrada ainda boa antes de consolidacao.

## 9. Ambiente Regulatorio

### 9.1 LGPD (Lei 13.709/2018)

**Impacto direto no produto:**
- Dados processuais sao publicos, mas contem dados pessoais (CPF, enderecos)
- Art. 7o, inciso VI permite tratamento para exercicio de direitos em processos judiciais
- SaaS de monitoramento atua como operador (Art. 5o, VII) — precisa de DPA com clientes
- Exige: politica de privacidade robusta, controles de acesso, logs de auditoria, criptografia

**Risco regulatorio:** Medio. Atividade tem amparo legal, mas armazenamento/revenda de dados de partes pode ser questionada.

### 9.2 CNJ — Resolucoes

- **Resolucao 331/2020 (DataJud):** Repositorio nacional de dados processuais. API publica gratuita.
- **Resolucao 335/2020 (PDPJ-Br):** Padronizacao de sistemas — implementacao ainda incompleta em varios tribunais.
- **Portaria CNJ 119/2021:** Regulamenta acesso a API publica do DataJud.

**Implicacao estrategica:** API do DataJud e ativo estrategico subestimado. Integracao com DataJud e diferencial de compliance e estabilidade.

### 9.3 OAB

- Responsabilidade do advogado sobre prazos cria demanda por SLA confiavel
- Credibilidade com OAB (eventos, endosso) e sinal forte de confianca

## 10. Oportunidades Estrategicas

**Oportunidade 1: API Processual B2B de Alta Qualidade**
- Nenhum player oferece API com SLA garantido, cobertura 90%+ tribunais, documentacao enterprise
- Potencial: R$ 5-15M ARR em 3-4 anos
- Risco: Custo de manutencao de cobertura permanente

**Oportunidade 2: Freemium para Autonomos como Topo de Funil**
- 400K autonomos como porta de entrada para viralidade
- Potencial: 50.000-100.000 usuarios free criando brand awareness
- Risco: CAC alto sem viralidade; custo de infra por usuario free

**Oportunidade 3: Monitoramento + Analitica (jurimetria basica)**
- Dashboards analiticos diferenciam do monitoramento puro
- Potencial: Premium tier 30-40% acima do preco basico
- Risco: Complexidade aumenta tempo de go-to-market

## 11. Recomendacoes Estrategicas

### 11.1 Go-to-Market

**Segmento prioritario:** Escritorios de pequeno e medio porte (5-50 advogados) com alto volume processual em direito trabalhista, previdenciario e civel.

**Raciocinio:** Sweet spot de ticket (R$ 300-1.200/mes), velocidade de decisao (2-4 semanas) e volume de mercado (~43.000 escritorios).

**Canal primario:** Digital-led (SEO juridico, conteudo, eventos AB2L/Fenalaw, parcerias com OABs seccionais).

**Parceiros estrategicos:**
- Sistemas de gestao juridica (Projuris, eLaw) como integradores da API — modelo B2B2B
- OABs seccionais (SP, RJ, MG) para programas de adocao tecnologica
- AB2L para credibilidade de mercado

### 11.2 Precificacao Recomendada

**Modelo:** Hibrido — Assinatura por volume de processos + creditos para API

| Plano | Preco | Perfil | Processos | Rationale |
|---|---|---|---|---|
| Free | R$ 0 | Autonomo | ate 30 | Aquisicao e ativacao; funil de brand |
| Solo | R$ 49/mes | Autonomo ativo | ate 200 | Abaixo do threshold de dor |
| Studio | R$ 199/mes | Escritorio 2-10 adv | ate 1.000 | Sweet spot de margem e conversao |
| Pro | R$ 499/mes | Escritorio 11-50 adv | ate 5.000 | Alto valor, integracao basica |
| Enterprise | Custom | 50+ adv, corporativos | Ilimitado | Contrato anual, SLA formal |

**Metrica de valor:** Numero de processos monitorados (intuitiva para o advogado, escala naturalmente).

**Complemento:** Creditos para consultas avulsas via API (pay-per-use) para integradores.

### 11.3 Mitigacao de Riscos
