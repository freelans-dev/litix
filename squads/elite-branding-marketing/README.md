# Elite Branding & Marketing Squad

**Version:** 3.0.0
**Slug:** `elite-branding-marketing`

A full-service branding and marketing agency squad for the Synkra AIOS framework. Composed of 9 specialized agents, each modeled after a legendary practitioner in branding, design, copywriting, or marketing.

## Agents

| Persona | Reference | Role |
|---------|-----------|------|
| Neumeier | Marty Neumeier | Brand Visionary -- Deep Discovery, ZAG, Golden Circle |
| Haviv | Sagi Haviv | Logo and visual identity |
| Scher | Paula Scher | Brand Book, complete visual system |
| Bierut | Michael Bierut | Voice Guide, manifesto, tone of voice |
| Ogilvy | David Ogilvy | High-conversion copy and campaigns |
| Godin | Seth Godin | Content strategy and editorial calendar |
| Kotler | Philip Kotler | Marketing plan, KPIs, performance |
| Bauhaus | -- | Visual asset production |
| Ive | Jony Ive | Brand experience QA and audit |

## Pipeline

```
Neumeier -> Haviv -> Scher -> Bierut -> Ogilvy -> Godin -> Kotler -> Bauhaus -> Ive
```

Each agent's output feeds directly into the next. The pipeline runs sequentially from brand strategy discovery through final QA audit.

## Skills

This squad uses 5 dedicated skills:

- **brand-strategy-brief** -- Strategic brief template (11 sections)
- **brand-book-generator** -- Brand Book template and process (10 sections)
- **copy-frameworks** -- AIDA, PAS, BAB, 4Ps library with channel templates
- **content-calendar** -- Editorial calendar and content strategy
- **brand-qa-checklist** -- QA checklist by deliverable type

## Usage

Activate the full squad:

```
@squad activate elite-branding-marketing
```

See `squad.md` for the complete definition, pipeline details, and governance rules.
